# MEP Connector UI Preview

## Tab: Licorp
### Panel: Modify

```
┌─────────────────────────────────────────────────────────────┐
│ Modify                                                      │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│ [Kết Nối ▼]  [Nâng hạ    ] [Copy/Move ] [Tăng giảm  ]      │
│              [cao độ    ]              [chiều dài  ]       │
│                                                             │
│              [Aline Đối ] [Aline      ] [Up&Down   ]       │
│              [Tương    ] [Hanger    ]                      │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

### SplitButton Dropdown Menu:
```
Kết Nối ▼
├── Kết Nối Hai Đối Tượng (Default)
├── Kết Nối Move Hai Đối Tượng Lại
├── Disconnect  
└── Kết Nối Aline Đối Tượng
```

### Button Functions:
- **Kết Nối**: Dropdown với 4 chức năng kết nối
- **Nâng hạ cao độ**: Điều chỉnh độ cao MEP elements
- **Copy/Move**: Copy hoặc di chuyển elements
- **Tăng giảm chiều dài**: Thay đổi chiều dài pipes/ducts
- **Aline Đối Tương**: Căn chỉnh elements
- **Aline Hanger**: Căn chỉnh hangers
- **Up&Down**: Di chuyển lên/xuống
